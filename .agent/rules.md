# Regras do Agente

- **Idioma:** Sempre mostrar tudo em Português do Brasil (pt-BR). Falar com o usuário e gerar conteúdos apenas em pt-BR.
- **Verificações de Navegador:** Não realizar verificações automáticas via browser (screenshots, testes de fluxo ou cliques) a menos que o usuário solicite especificamente uma ação de teste.
- **Git/Commits:** Realizar `git commit` ou `git push` apenas quando o usuário solicitar explicitamente. Não realizar commits automáticos após alterações de código.
